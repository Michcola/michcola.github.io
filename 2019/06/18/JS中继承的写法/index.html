JS中继承的写法

Posted on 2019-06-18

  1. 使用 prototype 如何继承

  2. 使用 class 语法糖如何继承

  3. 上面两个方法的区别

ES5中：

利用借用构造函数实现 实例属性和方法的继承 ；
利用原型链或者寄生式继承实现 共享的原型属性和方法的继承 。

ES6中：

利用class定义类，extends实现类的继承；
子类constructor里调用super()（父类构造函数）实现 实例属性和方法的继承；
子类原型继承父类原型，实现 原型对象上方法的继承。
